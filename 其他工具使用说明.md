# 办公工具箱使用说明

本文说明工具箱中默认展示的五项工具：背景资源打包、设备识别、ENI 查询下载助手、减速比计算和进制转化。

## 通用说明

- 在首页选择工具卡片进入相应页面；工具页顶部的“工具总览”可返回首页。
- 所有计算、图片转换和文件生成均在浏览器中完成。关闭或刷新页面会清空当前未下载的内容。
- 建议使用最新版 Chrome 或 Edge。下载文件通常保存到浏览器默认的“下载”目录。

## 背景资源打包

**用途**：将设备所需的 Logo、开机图、升级背景、二维码和关于信息转换为指定格式，并打包下载。

### 操作步骤

1. 在需要替换的资源卡片中点击“选择图片”。不需要替换的资源无需选择。
2. 选择图片后，系统会等比例缩放并居中填充到白色背景，自动转换为 PNG。升级程序背景需先选择适用版本：2207 或 2403。可在卡片中核对预览和文件名；需要重选时点击“更换图片”。
3. 如需生成关于信息，勾选“启用”。可以逐行输入，或将 1 至 9 行文本粘贴到输入框中。
4. 每条关于信息可设置左对齐或居中；`[ED]` 表示二维码下一行文字，最多只能勾选一行。
5. 确认底部“将被打包的文件”列表后，点击“下载 background.zip”。

### 生成文件规格

| 资源 | 适用版本 | 输出文件 | 尺寸 |
| --- | --- | --- | --- |
| Logo | 全部 | `Logo.png` | 145 x 60 像素 |
| 开机图片 | 全部 | `StartImage.png` | 800 x 600 像素 |
| 升级程序背景 | 2207 | `SoftUpdatingBackground.png` | 800 x 600 像素 |
| 升级程序背景 | 2403 | `SoftwareUpdatingBackground.png` | 800 x 600 像素 |
| 二维码 | 全部 | `QR.png` | 145 x 145 像素 |
| 关于信息 | 全部 | `Info.txt` | 固定 9 行、UTF-8 编码 |

### 注意事项

- 仅可选择图片文件。
- `Info.txt` 的粘贴内容最多 9 行；若粘贴内容已有 `[LE]`、`[MI]` 或 `[ED]` 标记，工具会自动识别。
- 至少选择一项图片或启用关于信息后，才可以下载压缩包。

## 设备识别

**用途**：根据 EtherCAT XML / ESI XML 或手动录入的信息，生成伺服与 IO 板设备配置文件 `slaveTypeLib.json`。

### 操作步骤

1. 页面初始包含一个设备条目。点击“添加伺服”或“添加 IO”可增加条目；每个条目右上角可切换设备类型或删除。
2. 优先点击“选择文件”导入对应的 XML。工具会尝试提取 Vendor ID、Product Code 和 PDO 索引等信息。
3. 补全或检查基本信息：标识名 `name`、中文名 `chineseName`、Vendor ID、Product Code。Vendor ID 支持输入十进制数或 `0x7DD` 形式的十六进制数。
4. 伺服设备可设置轴数与清编码器模式；IO 设备可设置 AD 比例，并在 PDO 配置中维护 AI、AO、DI、DO 条目。
5. 在右侧 JSON 预览中检查生成结果。可复制 JSON，或点击“下载 JSON” / “导出”下载 `slaveTypeLib.json`。

### PDO 配置说明

- AI、AO：每项设置 `bitnum`、`index` 和 `subindex`。
- DI、DO：除上述字段外，还需设置 `portnum`。
- XML 解析结果应与设备厂商资料核对；对于特殊 PDO 或非标准设备，应以实际 ESI 文件和控制器配置要求为准。

### 辅助换算

页面右侧的“参数转换”支持 Vendor ID 的十六进制与十进制双向换算。选择方向，输入数值后即可复制结果。

## ENI 查询下载助手

**用途**：在工具箱内打开 ENI 查询下载服务，也可单独打开该服务页面。

### 操作步骤

1. 进入工具后等待状态显示“已连接”。
2. 在内嵌页面中按 ENI 服务自身的流程完成查询和下载。
3. 页面右上角提供以下操作：
   - 刷新：重新加载 ENI 服务。
   - 全屏：在新窗口打开服务。
   - 外部打开：在独立标签页打开服务。

### 注意事项

- 该工具需要访问外部 ENI 服务，因此网络连接和服务可用性会影响使用。
- 内嵌页面约 6 秒未能响应时，会自动跳转到 ENI 服务的独立页面；这是正常的兼容处理。

## 减速比计算

**用途**：根据示教器显示值与实际测量值，计算机器人关节或直线轴的实际减速比。

### 操作步骤

1. 选择计算类型：
   - **角度**：适用于旋转关节，输入单位为 `deg`。
   - **直线**：适用于直线轴，输入单位为 `mm`。
2. 输入原始减速比 `a`、示教器显示的角度/距离、实际转动角度/移动距离。
3. 页面自动显示实际减速比 `b`、修正系数和相对偏差。
4. 点击“复制结果”可复制计算值及计算类型；点击重置图标可清空输入。

### 计算公式

- 角度模式：`b = a x (|theta| / |phi|)`
- 直线模式：`b = a x (|L1| / |L2|)`

其中 `a` 为原始减速比，`b` 为实际减速比。直线模式下两次距离必须使用相同单位。

### 注意事项

- 原始减速比必须大于 0，显示值和实测值不能为 0。
- 当显示值与实测值方向相反时，页面会提示方向不一致；当前计算仍按位移幅值进行。应先确认测量方向是否正确。

## 进制转化

**用途**：在二进制、十进制和十六进制之间转换整数。

### 操作步骤

1. 选择输入进制：自动识别、二进制、十进制或十六进制。
2. 输入整数后，页面会同时显示三种进制的结果。
3. 点击每个结果卡片右上角的复制按钮可复制对应结果；点击重置图标可清空输入。

### 输入规则

- 支持正数和负数，可使用 `+` 或 `-` 号。
- 输入中的空格和下划线会被忽略。
- 自动识别规则：以 `0b` 开头视为二进制；以 `0x` 开头或包含 A-F 字母视为十六进制；其余视为十进制。
- 二进制仅允许 `0` 和 `1`；十六进制允许 `0-9` 与 `A-F`，不区分大小写。
- 工具使用大整数计算，可处理超过普通 JavaScript 数值安全范围的整数。
