# 一寸照片生成器

## 启动

```bash
npm run dev
```

如果 3000 端口被占用：

```bash
PORT=3002 npm run dev
```

启动后电脑打开 `http://localhost:3002`，手机和电脑连同一个 Wi-Fi 后打开终端里打印的 `LAN` 地址，例如：

```text
http://192.168.3.7:3002
```

## 使用

1. 手机上打开页面。
2. 点“手机拍照”拍摄或选择照片。
3. 拖动照片并调整缩放。
4. 选择白底、浅灰底、蓝底或红底。
5. 点“生成一寸照”导出高清版 `600x840`，或者点“生成标准版”导出 `295x413`。
6. 点“导出图片”下载 PNG。

当前版本优先保证本地快速可用：支持拍照/导入、裁切、换画布底色、一寸电子照导出。自动抠背景可以后续接入模型服务增强。
